== Installation ==

1. Unpack archive with plugin
2. Rename the folder to /translate/
3. Copy the entire /translate/ directory into your /plugins/ directory.
4. Open /plugins/translate/config/translate.php and set the appropriate settings
5. Activate the plugin.
